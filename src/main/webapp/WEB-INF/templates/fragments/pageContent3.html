Over Onze Opleiding

Wij zijn een karaktervolle groep die vanuit een passie voor softwareontwikkeling er voor gekozen heeft om een intensief traject Java-development te doorlopen. Deze cursus wordt georganiseerd door Realdolmen in samenwerking met VDAB.


Wat kunnen de Java Potentials allemaal?

Zoals eerder vermeld werd, is dit een uitzonderlijk intensieve en uitgebreide opleiding. We zijn niet alleen bekend geworden met Java, maar hebben ook allerhande development tools leren gebruiken. Daarnaast werden we ook begeleid in het ontwikkelen van allerlei loopbaanvaardigheden. Naast het doorlopen van het traject, verkennen onze cursisten ook hun eigen specifieke ontwikkelings-gerelateerde interesses verder in hun vrije tijd. We hebben veel gedeelde kennis, maar ieder van ons heeft een persoonlijke waaier aan vaardigheden die daarop voortbouwt. Om even op te lijsten wat we allemaal in deze cursus hebben geleerd/gaan leren:

  • Java
  • Maven
  • Spring
  • JPA / Hibernate
  • JavaScript
  • TML & CSS
  • SQL
  • Docker
  • Jenkins


Wat zijn de doelstellingen van Java Potentials?

Wij kijken uit naar dit evenement als een opportuniteit om: enerzijds een job en werkomgeving te vinden die bij onze vaardigheden en gemotiveerde instelling past, en anderzijds om contacten te leggen binnen de industrie om onze carrières een goede start te geven. We hopen dat u net zo enthousiast bent over onze jobdag als wij zijn en kijken er naar uit om u te ontmoeten op ons ‘Java Potentials’ evenement
