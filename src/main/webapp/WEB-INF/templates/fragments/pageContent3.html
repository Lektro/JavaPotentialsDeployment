Over Onze Opleiding

Wij zijn een dynamische groep van jonge personen die, vanuit een passie voor softwareontwikkeling, in juni gestart zijn met een intensieve opleiding tot Java ontwikkelaar. Deze cursus wordt georganiseerd in samenwerking met Cevora en VDAB.


Wat hebben wij te bieden?

Zoals eerder vermeld werd, is dit een intensieve en uitgebreide opleiding. We verwerven een sterke basis in Java, en leren ook allerhande development tools gebruiken. Daarnaast worden we ook begeleid in het ontwikkelen van een aantal relevante soft skills op het vlak van communicatie en samenwerking. Doordat iedere deelnemer ook buiten de gemeenschappelijke lessen zijn specifieke interesses binnen development verder verkent, heeft ieder van ons ook een persoonlijke waaier aan vaardigheden ontwikkeld. Hieronder lijsten we even op welke modules in onze cursus aan bod komen.

  • Maven
  • Spring
  • JPA / Hibernate
  • JavaScript
  • TML & CSS
